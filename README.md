Geo Project 🌍
Образовательное приложение на Django для изучения географии мира


Функционал:

- Просмотр списка стран
- Добавление и редактирование стран
- Викторина по географии


Инструкция (как использовать):

1. Клонируйте репозиторий
2. Перейдите в папку репозитория при помощи cd
3. Введите команду: python manage.py runserver
4. Откройте в браузере: http://127.0.0.1:8000/
